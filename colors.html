Colors in CSS

    Colors in CSS can be specified by the following methods:
  • Hexadecimal colors
  • RGB colors
  • Predefined/Cross-browser color names
  • RGBA colors
  • HSL colors
  • HSLA colors

Hexadecimal Colors

  • A Hexadecimal color is specified with: #RRGGBB, where RR(red), GG*(green), BB(blue) Hexadecimal integers specify the components of the colors. All values must be between 00 and FF,
  • Foe Example, the #0000ff value in rendered as blue, because the blue component is set to its highest value (ff) and others are set to 00.

RGB Colors

  • An RGB color value is specified with the rgb() function, which has the foloowing syntax: rgb(red, green, blue)
  • Each parametre(red, green, blue) defines the intensity of the colors and can be an integer between 0 and 255 or a percentage value (from 0% to 100%)
  • For example, the rgb(0,0,255) is rendered as blue, because the blue parametre is set to its highest value(255) and the others are set to 0.

Predefined/Cross-browser color

  • 140 color names are predefined in the HTML and CSS color specification.
